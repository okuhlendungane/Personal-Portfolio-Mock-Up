# 🌐 Portfolio Website (Mock Project)

This is a mock portfolio website built using HTML and CSS. The project focuses on creating a clean, responsive, and visually appealing personal website layout.

## 🚀 Features
- Responsive design for different screen sizes
- Structured layout with multiple sections (Home, About, etc.)
- Custom styling using CSS
- Clean and simple UI design

## 🛠️ Technologies Used
- HTML
- CSS

## 🎯 Purpose of the Project
The goal of this project was to practice front-end development skills, especially:
- Page layout and structure
- Responsive design techniques
- Styling and positioning elements using CSS

## 📌 What I Learned
- How to structure a multi-section webpage
- Improving responsive design using CSS
- insert images and grid layouts
- Handling layout issues (e.g. overlapping elements)
- Writing cleaner, more organized code

## 🔧 Future Improvements
- Add JavaScript for interactivity
- Improve animations and transitions
- Connect to a backend or make it fully functional
- Deploy the website online
- improve grid layout
